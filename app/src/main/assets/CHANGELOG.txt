Changelog for Quick App Launcher

List of changes, newest changes first.

Version 1.0.3
 - Removed permission for in app purchases as this was never used.

 * Bugfix: There was a NPE in some cases when setting up the UI.
 * Now also non-touch devices can download the app.
 * Bugfix: Recently installed and updated lists weren't updated if app wasn't
   running.

 + Initial release.
